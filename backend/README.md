## 项目说明

这是为 Flutter 毕设配套的 NestJS 后端，提供登录、成绩、考试倒计时的 REST API。

### 本地启动

1. 复制环境变量

```bash
cp .env.example .env
```

2. 安装依赖并生成 Prisma Client

```bash
npm install
npx prisma generate
```

3. 运行数据库迁移（首次）

```bash
npx prisma migrate dev --name init
```

4. 启动服务

```bash
npm run start:dev
```

### Flutter 端接入要点

- 所有接口统一前缀：`/api`
- 认证接口：
  - `POST /api/auth/register`
  - `POST /api/auth/login`
  - `GET /api/auth/me`
- 业务接口需要 `Authorization: Bearer <token>`：
  - `GET /api/grades`
  - `POST /api/grades`
  - `PUT /api/grades/:id`
  - `DELETE /api/grades/:id`
  - `GET /api/exam-countdowns`
  - `POST /api/exam-countdowns`
  - `PUT /api/exam-countdowns/:id`
  - `DELETE /api/exam-countdowns/:id`
- 资讯接口（公开，无需登录）：
  - `GET /api/news?category=notice&limit=20&offset=0`

### 数据库准备（资讯表）

新增了 `campus_news` 表模型，迁移步骤：

```bash
npx prisma generate
npx prisma migrate dev --name add_campus_news
```

可选：在 Supabase SQL Editor 或数据库工具中插入测试资讯数据，字段与接口返回一致。
